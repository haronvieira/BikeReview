# BikeReview - Sistema de Revisão de Bicicleta

## Descrição
O BikeReview é um sistema simples para gerenciar revisões períodicas de bicicletas. Ele permite que os usuários registrem e acompanhem manutenções realizadas em suas bicicletas, garantindo que elas estejam em boas condições de uso.

## Como Usar
Bicicleta sempre limpa: Sujeira gera problemas no conjunto de transmissão e freios, portanto, manteha o equipamento sempre limpo e lubrificado
Regulagem das marchas: Manter a regulagem das marchar é fundamentalpara evitar acidentes e também para que a corrente não saia das catracas.
Conferir os pneus: Pneu desgastado gera acidente, além também, os mesmo devem estar calibrados para manter o boa performace.
Verificação de freios: É fundamental manter a regulagem e a troca das pastilhas de freios, quando esgotam-se a vida útil.
Alinhamento: Para ter a melhor experiência é de extrema importância que mesa, guidão e cambios, estejam alinhados.
